Navigation

Navigation on this page is available through the Navigation Bar above the main content.

  • Home: will take you to the dashboard.
  • News: will take you to a page where you can review news stories by month and type – Industry or Clubwatch.
  • Manage Reports: will allow you to review previous Venue Activity Reports by month. This is where you can select Venue Activity Reports to go into the details.
  • Manage Venue: (Club owners only) will allow a user who is a venue owner type to view and edit details for their venue.
  • Create Report (venue staff) is a short-cut available to lower-authority level users in the system. If this option is available to you, it will take you directly to the Venue Activity Report Creation page.

Dashboard

On arriving at the dashboard screen, you will see Industry News and Clubwatch Announcements pertaining to your venue’s region. Below the announcements there is a summary of the last 3 Venue Activity Reports for your venue. The left side of the screen gives you a graphical depiction of the last 45 days attendance and incidents. You may move the mouse pointer over the graphs to see the specific values. The section headings are links directly to a page with information relating to that subject.

News

This page will display Clubwatch Announcements for the current month by default. If you came to this page by clicking a heading section on the dashboard page it will show news stories for the type you clicked. You may select news stories of a different type or for a different month by making choices from the News Reel filter at the top of the content area and pressing the Get button.

Manage Reports

The page will display Venue Activity Reports for the current month by default. You may select Venue Activity Reports for a different month by making choices from the date filter at the top of the content area and pressing the Go button. If you click on the Date of a specific Venue Activity Report you will open the page for that report. This allows you to view and modify the report, including incidents. The Create New Report button at the very top of the content area will take you to a page where you can create a brand new Venue Activity Report. There is only one VAR per night but there can be many incidents that may occur during that night.

  1. Fill-out all fields, and then add new incident reports to the VAR when needed. You can always come back and add new reports by pressing the Add new Incident button.
  2. Once inside the Incident Report forum, fill out all appropriate fields pertaining to that incident. Adding persons in interest can be done by pressing the Add a Person button. You can also print a hard copy of the report at any time by pressing the Print button.
  3. Once in Add a person fill out all necessary fields and save entry/ Delete entry if incorrect.

Venue Activity Reports

This page allows you to create a new Venue Activity Report. Some fields are already filled in with default values.

  • Submitting for the night of: will allow you to pick a date from the drop down. The system will only allow you to have one report for any specific date.
  • Event: allows you to input the name of a special event for that night, if you wish. The default value is Reg. Op. for regular operations.
  • Total Nightly Attendance: allows you to input a number for your nightly attendance.
  • Name of Shift Supervisor will accept any name you choose to input, even people who are not in the system. This name will appear on any subsequent incident reports for that night. The default name is the current logged in user.
  • Security Checklist Completed takes a yes or no value, indicating whether the security checklist was completed for the night of the report. Security Checklists are available for download from Clubwatch.
Button Action
Cancel Allows you to exit the Create page without creating a new Report
Add New Incident Saves your new Activity Report and takes you to the Incident Report page
Finished Saves your new Activity Report and takes you back to the report list
Update Saves changes to an existing Activity Report
Delete Deletes this Activity Report and returns you to the report list
Done Exits the view/modify page without saving

Incident Reports

This page lets you view incidents reports, make changes, or create new incident reports. The following fields are automatically filled in based on the Venue Activity Report:

  • Venue Name:the name of your establishment
  • Venue Address: the address listed for your establishment
  • Venue City & Province: the city and province in which your establishment is located
  • Report Date: the date of the Venue Activity Report this incident is saved to
  • Event Occurring: the event listed on the Venue Activity Report this incident is saved to
  • Supervisor on Shift: the supervisor name on the Venue Activity Report this incident is saved to
  • Form Completed by: the name of the user currently logged in
  • Venue Contact:the person listed as your establishment's liaison (can be changed under venue management)

Persons of Record

In order to save a record of a person, you must at the very least fill out the Notes field.

  • Name: will accept any name given
  • Involvement: Allows you to specify this persons relation to the incident
  • Phone: will accept a 10 digit number
  • License: will accept an entry, up to 25 characters
  • Notes: the reason this person is being recorded must be entered
Save Saves any changes made
Cancel Exits without saving, returning to the Incident Report page

Creating a new Incident Report

To save an incident report, you must fill out the time field and provide a description of the incident. The damages field is optional and you may wish to specify a new level of severity. Press the Save button at the bottom of the page to exit and return to the Venue Activity Report View or press Add a Person to save and go to the Person form where you can attach information about persons associated with this incident.
