Manage News

On this page the administrator is able to create, modify and delete news items. First a month is selected then the news articles for that month will appear with associated editing icons.

  • The page/green plus symbol icon is to create a new news article.
  • The folder icon is used to save the news to archives.
  • The paper/pencil icon is used to modify a previously created new item.
  • And the red 'X' is used to delete a news article.
  • Once in a new or previous report the admin will fill out necessary fields and save the news article to the database.

Manage Region

This page will display a list of regions with options to create a new region or modify an existing one.

  • To create a new region, select the new region button. Then enter the new region number, region name and submit.
  • To modify an existing region, select the modify button. Then enter the new region name and submit.

Manage venues

This page allows you to see all venues and make modifications them or create new venues. In addition, it allows you to disable or enable a venue, denying or granting users access to that venue. Changing a venues status: simply select active or deactive from the selection box associated with the venue. A prompt will alert you that the venue status has been successfully updated. To modify a venue, click the modify button associated with the venue. To create a new venue, click the new venue button at the top of the page.

Venues page

Allows you to make changes to a venue's information. Note that when creating a new venue, you must fill out at least a venue name before being able to create or add users to the venue. Fields:

  • Name: The venues names. This field is mandatory.
  • Unit: The venues unit number.
  • Address: The venues address.
  • City: The venues city
  • Province: The venues province.
  • Postal Code: The venues postal code.
  • Region: The region the venue is located in. Used to show news.
  • Phone: The venues phone number.
  • Contact: The venues primary contact.
  • Create Owners: Whether or not the venue owner is allowed to create additional owners. If no, Clubwatch will have to add additional owners.
You can make changes to user information in the table at the bottom of the page. Note that deleting a user only removes them from that venue, they will still have access to any other venues they may be a part of. Also note that changing their username will change the username that they use to log on to all venues they may be associated with. Any changes made will only be saved when you click on the update user button for a particular user.

Create user button: Brings up a form to create a new user. The user name for the new user must be unique, and the password must be at least 7 characters long. It is recommended that the user change their password when they first log on.

Join user button: Allows you to join or re add an existing user to a venue.
